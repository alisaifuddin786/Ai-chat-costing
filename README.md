# Run and deploy your Next.js App Router project

This repository is now a full **Next.js (App Router)** application and is ready to deploy on Vercel.

## Run Locally

**Prerequisites:** Node.js 18+

1. Install dependencies:
   `npm install`
2. Set `NEXT_PUBLIC_GEMINI_API_KEY` in `.env.local`
3. Run the app:
   `npm run dev`

## Deploy on Vercel

1. Push this repository to GitHub/GitLab/Bitbucket.
2. Import the project in Vercel.
3. Add `NEXT_PUBLIC_GEMINI_API_KEY` to Vercel Environment Variables.
4. Deploy.
